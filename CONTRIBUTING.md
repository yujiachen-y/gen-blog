# Contributing

Thanks for helping improve this project!

## Development setup

```
npm install
```

## Common scripts

- `npm run generate -- <vaultDir> [outputDir] [--site-url <url>]`
- `npm run lint`
- `npm run format`
- `npm run format:check`

## Guidelines

- Keep changes minimal and focused.
- Avoid adding new config/env vars unless necessary.
- Preserve architecture boundaries (generator vs theme).
- Run formatting before opening a PR.

## Reporting issues

Please include:

- Steps to reproduce
- Expected vs actual behavior
- Node.js version
- Sample input (redacted if needed)
